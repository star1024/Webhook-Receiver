Webhook Receiver + Event Log

Receive webhook payloads at /webhook, inspect them in real time, and reset the log during demos.

Endpoints

Use these directly in Postman, curl, or another service.

POST /webhook

Receive any JSON payload and append it to the event log.

GET /api/events

Fetch the latest captured events.

GET /api/health

Simple health check for local demos.
